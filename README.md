## Stack

- Pure HTML5, CSS3, JavaScript (no frameworks, no build step)
- Google Fonts: Inter + Fira Code
- Hosted on GitHub Pages

## Customizing

1. Open `index.html` and search for `<!-- ` comments — every placeholder is marked
2. Replace `YOUR_USERNAME` with your GitHub username throughout
3. Add your projects in the **Projects** section (copy the `<article class="project-card">` block)
4. Drop your resume PDF at `assets/Benjamin_Winter_Resume.pdf`
5. Add a profile photo at `assets/profile.jpg` and swap out the placeholder div in the About section

## Deploying to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from a branch`, branch `main`, folder `/ (root)`
4. Your site will be live at `https://bwinter1248.github.io/Bens-GitHub-Portfolio`
